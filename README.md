# Складская микро-система на чистом JavaScript

Консольное приложение, симулирующее работу товарного склада. Проект написан без использования HTML/DOM и сосредоточен исключительно на архитектуре данных и внутренней логике приложения.

### 🎯 Цель проекта
Отработка объектно-ориентированного подхода в JavaScript: создание методов объектов, динамический доступ к свойствам через квадратные скобки и практическое применение контекста вызова функций (`this`).

### 🚀 Ключевые особенности реализации
* **Безопасный обход объекта (`for...in`):** Реализована фильтрация данных с проверкой типов (`typeof this[key] === "function"`). Это защищает математические расчеты от сбоев при наличии методов внутри объекта.
* **Надежная валидация ключей (`in`):** Метод приемки товара защищен от добавления несуществующих позиций, что предотвращает загрязнение базы данных склада.
* **Использование контекста (`this`):** Логика изменения остатков и подсчета стоимости завязана на динамическом обращении к свойствам текущего объекта.
* **Финансовый учет:** Внедрен сквозной подсчет глобальной прибыли от продаж со своевременным логированием результатов в консоль.
